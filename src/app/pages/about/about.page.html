Sobre o Aplicativo

Quem Somos?

Jackeline Ferreira é a responsável pela documentação, Leonardo Lima pelo back-end e Pedro Gregorio pelo front-end.

Nossa Proposta

Desenvolver um aplicativo mobile com a intenção de auxiliar mulheres em possíveis situações de risco.

Tecnologias Utilizadas

Para o desenvolvimento do front-end, foram utilizados os frameworks Angular e Ionic com Typescript de linguagem, já para o back-end, Java foi a linguagem escolhida junto o framework Spring Boot e MySQL para o banco de dados.
